    ostk.simulation.Simulator¶

    class Simulator(self: ostk.simulation.Simulator, environment: ostk.physics.Environment, satellites: list[ostk::simulation::Satellite])¶

    Bases: pybind11_object

    Main simulation orchestrator managing satellites and environment.

    The Simulator coordinates the overall simulation, managing a collection of satellites, the physics environment, and the simulation time. It provides methods to step forward in time and manage satellite lifecycles.

    Create a Simulator instance.

    Parameters:

    • environment (Environment) -- The physics environment (celestial bodies, gravity, etc.).

    • satellites (list[Satellite]) -- Array of satellite objects to simulate.

    Returns:

    The simulator instance.

    Return type:

    Simulator

    Example

    >>> simulator = Simulator(
...     environment=environment,
...     satellites=[satellite1, satellite2]
... )
